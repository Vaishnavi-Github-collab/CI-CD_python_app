# CI-CD_python_app
## CI/CD Pipeline Implementation

This project implements a complete CI/CD pipeline using GitHub Actions.

### Features:
- Continuous Integration (CI) for automated testing
- Performance Testing using K6
- Security Testing using OWASP ZAP
- Continuous Deployment (CD) using Docker

### Tools Used:
- GitHub Actions
- Docker
- K6
- OWASP ZAP
- Python (Flask)

The pipeline runs automatically on every push to the main branch and deploys the application to a staging environment after successful tests.
